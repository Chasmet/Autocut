CutFlow

Découpe audio et vidéo par tranches fixes

Application maison

Découpe rapide, claire, agréable sur mobile

Choisis un fichier, sélectionne une tranche, puis télécharge chaque segment un par un avec des noms simples : Cut 1, Cut 2, Cut 3...

1. Importer un média

Audio ou vidéo

Aucun fichier sélectionné

2. Choisir la tranche

5, 10, 20 ou 30 secondes

Tranche choisie 5 secondes
Durée détectée -

3. Lancer la découpe

Le traitement se fait dans le navigateur

En attente d’un fichier 0%

4. Résultats

Chaque cut est téléchargeable séparément

Les segments apparaîtront ici après la découpe.

Important

Pour que l’application marche bien sur téléphone, utilise de préférence GitHub Pages. Ouvrir directement le fichier HTML dans le téléphone peut bloquer certains chargements du moteur vidéo/audio.
